# AlphaGSM Documentation

This folder is for people who want help running a specific server.

If you are not sure where to start, pick your server below and follow that guide.

## Server Guides

- [Minecraft Vanilla](servers/minecraft-vanilla.md)
- [Team Fortress 2](servers/team-fortress-2.md)
- [Counter-Strike: Global Offensive](servers/counter-strike-global-offensive.md)

Each guide tries to show:

- what you need first
- which commands to run
- what a real working example looks like

The guides also point to the smoke-test runners in `smoke_tests/`. Those scripts are the best real examples in the repository because they show a full working start-to-stop flow.
