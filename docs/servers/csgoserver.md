# csgoserver

`csgoserver` is an alias for [`counterstrikeglobaloffensive`](servers/counterstrikeglobaloffensive.md).

This alias points at the disabled legacy CS:GO app `740` module. Use
[`counterstrike2`](counterstrike2.md) for the current CS2 dedicated-server flow.

```bash
alphagsm myserver create csgoserver
```

See the [counterstrikeglobaloffensive](counterstrikeglobaloffensive.md) guide for legacy details.
