AlphaGSM changelog
==================

Current release notes should be updated in every PR that changes user-facing behaviour,
server support, CI, packaging, or workflow defaults.

Historical detail lives in git history. Keep this file focused on release-facing notes and
high-level milestones.

Historical highlights
---------------------

2014
- Initial repository created.

2015
- Built the first AlphaGSM CLI foundation, including command parsing, output multiplexing, and early Minecraft support.
- Renamed the project from sagsm/samanager to AlphaGSM and expanded early documentation.

2016
- Added downloader, settings, backup, activation, and update workflows that shaped the current lifecycle model.
- Expanded Minecraft variants and Steam-oriented support while stabilising the configuration system.

2017
- Continued Steam/public branch integration work and small maintenance merges.

2021
- Refreshed GitHub-facing documentation.

2024
- Resumed active maintenance on the live branch.

2026
- Accelerated platform and runtime work across Windows compatibility, Proton, Docker runtime families, docs/wiki automation, more commands, and broader game-module coverage.
- Hardened CI and release workflows with runtime image publishing, port-collision fixes, and tighter test workflows.
